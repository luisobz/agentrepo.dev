# Task 3: Layout tipo IDE y Task 4: FileTree & Tabs

## Descripción
La experiencia de usuario clave de AgentRepo: un entorno tipo VS Code en web.

## Criterios de finalización
- [ ] Layout con tres columnas responsivas: (Sidebar de ficheros, Área de pestañas/código, Metadatos a la derecha).
- [ ] Componente recursivo `FileTreeView` que renderiza carpetas expandibles y ficheros.
- [ ] Sistema de estado local (Zustand o Context) que mantiene qué ficheros están abiertos en `tabs` y cuál es el `activeTab`.
- [ ] Visor Markdown/Código integrado para pintar el contenido del fichero activo.
