# Task 3: Blog Público UI

## Descripción
Página pública para leer los posts. Diseño editorial, minimalista, centrado completamente en la lectura sin barras laterales ruidosas.

## Criterios de finalización
- [ ] Ruta `/blog` que lista los posts publicados por fecha descendente.
- [ ] Ruta `/blog/[slug]` que muestra el post completo procesado por el markdown-viewer.
- [ ] Estilos de tipografía (prose) limpios.
