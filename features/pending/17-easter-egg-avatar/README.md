# Feature 17: Easter Egg Avatar

## Meta
- **Número:** 17
- **Fase:** Fase 5 — Pulido y Lanzamiento
- **Objetivo detallado:** Implementar la lógica interactiva del Avatar en el frontend. Si el usuario hace click repetidamente (5 veces) sobre el avatar en la web, este cambiará sus estados visuales de emoción de forma secuencial y finalmente lo redirigirá a la sección privada del portfolio `/portfolio/luisbz`.
- **Dependencias:** 04-fundacion-avatar-system.

## Alcance
- Hooks de conteo de clicks persistentes en la sesión de frontend.
- Integración con el contexto global de animación del Avatar.
- Desbloqueo y persistencia en localStorage/sessionStorage.
