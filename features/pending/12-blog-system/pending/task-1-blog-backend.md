# Task 1: Backend de Blog (Posts)

## Descripción
Implementar el dominio, infraestructura y capa de aplicación para el micropost del blog. Similar al de Skills pero más sencillo, enfocado en título, slug, extracto, contenido y fecha de publicación.

## Criterios de finalización
- [ ] Entidad `Post` creada.
- [ ] `PostsRepository` y modelo de Prisma implementado.
- [ ] Casos de uso CRUD.
- [ ] Router tRPC de blog exportado.
